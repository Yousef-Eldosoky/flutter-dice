## How to Use  
1. **Select the Number of Dice:**  
   Choose to roll 1, 2, or 3 dice.  

2. **Roll the Dice:**  
   Tap the **Roll** button to generate your random dice results.  

3. **View the Results:**  
   The app displays the outcomes of the dice roll instantly.
